Taxi&Go - App de Transporte Público

Desarrollo Web Diseño UI/UX Aplicación Web Transporte

Taxi&Go es una aplicación web innovadora diseñada para facilitar el acceso al transporte público, enfocándose especialmente en el servicio de taxi en Tenerife.

Esta solución digital moderniza completamente la experiencia de solicitar un taxi, eliminando intermediarios y ofreciendo una plataforma accesible para la población joven. La app permite reservas instantáneas, opción de viaje compartido para mayor ahorro, y está diseñada específicamente para las necesidades de movilidad en la isla.

El proyecto fue desarrollado con tecnologías web modernas, diseño responsivo y una interfaz intuitiva que funciona perfectamente en dispositivos móviles y ordenadores.
